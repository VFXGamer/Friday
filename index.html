Friday V.A.

This project is inspired by Marvel's Iron Man/Tony Stark who created Friday V.A. after his first virtual assistant Jarvis.

View on GitHub Download .zip Download .tar.gz

Steps of Installation

First Step: Download the project file.

Second Step: Run the following command python -m pip install -r Requirements.txt

Note:- If there is any error while downloading any module then go to my github page and raise issue I will reply within 24hrs.

Third Step: Run the following command python FridayCode.py This will launch the file.

Note:- This Program will only work for WINDOWS
